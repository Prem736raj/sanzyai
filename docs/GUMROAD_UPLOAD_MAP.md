## Fast Upload Steps

1. Open Gumroad product editor for a pack.
2. In Content/Files, remove old delivery file.
3. Upload mapped `.zip` file from list above.
4. Save/update product.
5. Repeat for all 10 paid products.

## Verification Checklist (per product)

- Product title matches mapped file.
- New file name visible in Gumroad asset list.
- Old file removed (avoid buyer receiving outdated prompts).
- Test purchase/download opens the new prompt pack content.
