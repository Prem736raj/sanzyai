# Premium Prompt Quality Standard (All Packs)

Use this standard for every paid pack asset before publishing to Gumroad or any delivery platform.

## Minimum Standard Per Prompt

Every prompt must include these 6 blocks:

1. Problem solved
- One sentence describing the exact business/user pain.

2. Best for
- Specific buyer type, role, or use case.

3. Input variables
- All placeholders the buyer must replace.

4. Core prompt
- Detailed task instruction with clear context and boundaries.

5. Output format
- Exact deliverables required (sections, bullets, length, style).

6. Quality checks
- 3 to 5 checks that prevent low-quality output.

## Hard Rejection Rules

Reject any prompt that:
- Is under 120 words (unless it is explicitly a micro-format prompt).
- Has no clear output format.
- Uses vague instructions like "make it better" without measurable criteria.
- Has no target user/context.
- Cannot be reused across real projects with minor variable changes.

## Quality Scoring Rubric (0-5 each)

- Clarity: Is the task unambiguous?
- Practicality: Can buyer use output immediately?
- Specificity: Are constraints and variables defined?
- Conversion Utility: Does it connect to a real business outcome?
- Reusability: Can this work repeatedly across niches?

Minimum release threshold: 22/25 average per pack.

## Pack-Level Release Checklist

- At least 80% prompts in the pack pass 22/25+.
- At least 30% prompts are advanced templates (multi-step workflows).
- Include category coverage (strategy, execution, optimization, QA).
- Include "mistake avoidance" prompts in each pack.
- Include one quick-start guide showing best prompting workflow.

## Recommended Buyer-Friendly Prompt Structure

Template:

Problem solved: ...
Best for: ...
Inputs to replace: ...
Prompt: ...
Output format: ...
Quality checks: ...

## Important Note for This Codebase

- The website currently stores and renders sample prompts in `src/prompt-store.js`.
- Paid full-pack files are delivered externally (e.g., Gumroad assets).
- Keep your external paid assets synced with this quality standard to avoid buyer regret.
